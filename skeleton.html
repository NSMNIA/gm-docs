Skeleton

Deze pagina is een overzicht van het skeleton met gebruik van Git.

De bedoeling van het skeleton in Git is dat er altijd standaard componenten bij toegevoegd kunnen worden, dus stel er is een nieuwe navigatie gemaakt dan kan deze naar Github worden gepusht. Dan kan daarna de Platform Generator een nieuw platform aanmaken met de nieuwe standaard componenten. Dit geldt voor Front- en Back-end.
